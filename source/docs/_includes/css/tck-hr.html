Horizontal rules.

The HTML <hr /> tag is used for creating a horizontal lines to separate content.

Horizontal rules aren't evil.

The horizontal rules defines a thematic break in an HTML page. Some examples of thematic breaks that can be marked up using the hr element include a scene change in a story, or a transition to another topic within a section of a reference book.

There are many ways to separate visual and semantically. With <section> tags and also with css attributes like border lines and margins. But sometimes, a horizontal rule can be a perfect solution.
